How to play:

Remove a pair of identical tiles at a time. The goal of the game is to remove as many tiles as possible. You can only select open tiles, which can be moved either left or right without disturbing other tiles.

If you get stuck, use the shuffling feature or the time-travel tree to jump back to any previous moment from where you can start over.
